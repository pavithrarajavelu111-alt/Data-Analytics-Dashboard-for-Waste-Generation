# Waste Generation Data Analytics Dashboard

This project focuses on analyzing and visualizing waste generation data to identify trends, patterns, and high-waste areas.

## Tools & Technologies
- Python (Pandas, NumPy)
- Tableau
- Excel / CSV datasets

## Key Features
- Area-wise waste distribution analysis
- Daily and weekly waste trend visualization
- Interactive dashboard using Tableau
- Data-driven insights for efficient waste management

## Outcome
The dashboard helps stakeholders understand waste generation patterns and supports better planning and decision-making.
